# runbook-templates

Public reference library of Ntropii runbook templates.

Each runbook is a self-contained example showing how a workflow type
is wired against the [`ntro` SDK](https://docs.ntropii.com). Read,
copy, adapt — but treat this repo as read-only.

Authoritative source lives at `ntropii-com/runbook-templates-internal`;
each commit here is a squashed publish via the
[publish-to-public](https://github.com/ntropii-com/runbook-templates-internal/blob/main/scripts/publish-to-public.sh)
script.

See [Ntropii docs](https://docs.ntropii.com) for context on what a
runbook is and how it fits into the platform.
